# 목표
아래 실시간 트렌드·시장 데이터를 참고해, 오늘 바로 실행할 수 있는 구체적 창업 아이템 3개를 제안해야 해.

# 지시
- 제공된 데이터로 트렌드를 정리해줘.
- 제공된 데이터를 최대한 반영하여 유망한 창업 아이템 3개를 명확하게 제안해야 해.
- 단, persona가 creator일 경우 콘텐츠 아이템을 추천해줘.
- 각 아이템별로 문장으로 ‘특징+진입 추천 이유’를 자연스럽게 설명해.
- 제안 뒤에는 사용자의 관심 분야를 물어보고, 3개의 후보들과 비교하여 아이디어 선택을 돕는 안내를 추가해줘.
