# 목표  
- 고객의 행동/구매 데이터를 기반으로 유의미한 고객 그룹(세그먼트)을 정의하고, 각 그룹에 맞는 CRM 자동화 전략을 제안해줘.  
- 사장님이 직접 활용할 수 있도록, 고객 세그먼트 정의부터 자동화 시나리오, 활용 채널까지 단계적으로 구성해줘.  
- 너무 추상적이지 않게, 실제 마케팅 툴이나 메시지 흐름까지 고려한 제안이면 좋아.

# 출력 포맷  
1. 주요 고객 그룹 정의  
고객 데이터를 기반으로 실질적으로 나눌 수 있는 대표적인 고객 그룹(예: 첫 구매 고객, 장바구니 이탈 고객 등)을 2~3개 제안해줘.

2. 각 그룹에 맞는 자동화 시나리오  
각 그룹에 대해 어떤 메시지를 언제, 어떤 방식으로 보낼 수 있을지 간단한 자동화 흐름 예시로 설명해줘.  
(예: 첫 구매 3일 후 감사 메시지 + 리뷰 요청 → 미작성 시 일주일 후 리마인드 메시지)

3. 활용 채널 제안  
각 자동화 시나리오에 적합한 채널(예: 이메일, 카카오톡, 푸시 알림 등)을 제안해줘.  
"이런 식으로 활용하시면 좋습니다" 같은 말투로 설명해줘.
